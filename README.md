# Shalfa 0.1
Shalfa is a simple tool that helps you with:
- Extract URLs.
- Find subdomains.
- Find some headers misconfiguration.

# Installation
```
git clone https://github.com/0xAbdullah/Shalfa.git
cd Shalfa 
pip3 install requirements.txt
```

# Usage
```python
python3 shalfa.py -u https://example.com
```
